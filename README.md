## Overview

A full-stack inventory management application built with React, Scss, Node.js, Express, and MongoDB. Features user registration, product management, inventory tracking, and transaction history.

---

## Features

- **User Registration**: Capture user email and full name.
- **Product Registration**: Capture SKU, product name, price, and initial quantity.
- **Inventory Management**: Increase or decrease product stock (prevent stock from dropping below zero).
- **Product Dashboard**: Display SKU, current quantity, and the timestamp of the last update.
- **Transaction Ledger**: Display a paginated list logging all historical stock changes.
 
---

## Technologies Used

- **Frontend**:
  - React.js (with React Router and Redux)
  - Sass for style
- **Backend**:
  - Node.js and Express.js
- **Database**:
  - MongoDB (via Mongoose)
